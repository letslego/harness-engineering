# Veritabanı Şeması

Bu klasörü, ajanların onları koddan tersine mühendisliğe gitmek zorunda
kalmadan inceleyebilmesi gereken üretilmiş ya da türetilmiş çıktılar için
kullanın.

## Kaynak

- Üretildiği yer: `[komut ya da kaynak yolu]`
- Son yenilenme: `YYYY-MM-DD`

## Notlar

- Üretilmiş bölümleri elle düzenlemeyin.
- Alttaki şema değiştiğinde bu dosyayı yeniden üretin.
