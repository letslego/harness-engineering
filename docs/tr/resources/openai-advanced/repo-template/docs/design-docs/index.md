## Bakım Kuralları

- Her tasarım dokümanının bir sahibi ya da güncelleme tetikleyicisi olmalıdır.
- Bayat dokümanları kaldırın ya da sürüklenmesine izin vermek yerine
  geçerliliğini yitirmiş olarak işaretleyin.
- Aktif yürütme planlarını bağımlı oldukları tasarım dokümanlarına bağlayın.
