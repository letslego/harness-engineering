# 高階儲存庫範本

當你想在真實儲存庫裡採用 OpenAI 風格、以代理為優先的文件面，而不再只靠最小執行環境時，就複製這套起步範本。

## 複製順序

1. 把 `AGENTS.md` 和 `ARCHITECTURE.md` 複製到儲存庫根目錄。
2. 複製整個 `docs/` 目錄樹。
3. 優先填寫 `docs/PRODUCT_SENSE.md`、`docs/QUALITY_SCORE.md` 與 `docs/RELIABILITY.md`。
4. 在 `docs/exec-plans/active/` 下加入第一份進行中的計畫。
5. 讓入口檔案保持簡短，並把細節導向連結文件。

## 這套範本優先處理什麼

- 可長期保存、位於儲存庫內的脈絡
- 漸進揭露，而不是單一巨大指令檔案
- 明確的計畫生命週期
- 隨時間累積的品質追蹤
- 讓代理與人都容易閱讀的邊界

這裡的每個檔案都只是起點。在你真的依賴它之前，先把佔位內容、範例與示範指令替換成專案本身的細節。
