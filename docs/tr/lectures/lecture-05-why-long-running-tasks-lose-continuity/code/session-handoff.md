# Oturum devri örneği

## Tamamlanan

- Markdown içe aktarma desteği eklendi
- Renderer'a temel bir belge listesi eklendi

## Bozuk veya doğrulanmamış

- `.md` için içe aktarma başarılı oluyor ancak büyük `.txt` dosyaları için başarısız oluyor
- Uygulama başlıyor, ancak detay görünümü bağlanmadı

## Sonraki en iyi adım

- `.txt` içe aktarma yolunu düzeltin
- İçe aktarmayı uçtan uca doğrulayın
- Ardından belge detay panelini ekleyin
