# AGENTS.md

## Buradan başlayın

- `docs/ARCHITECTURE.md` dosyasını okuyun
- `docs/PRODUCT.md` dosyasını okuyun
- Uygulamayı başlatmak için `npm run dev` kullanın
- İşi tamamlanmış olarak işaretlemeden önce `npm run check` kullanın

## Katı kurallar

- `docs/ARCHITECTURE.md` dosyasını okumadan Electron main/preload/renderer
  sınırlarını değiştirmeyin
- Doğrulama yapmadan bir özelliği tamamlanmış olarak işaretlemeyin
- Sonraki oturum için temiz bir durum bırakın
