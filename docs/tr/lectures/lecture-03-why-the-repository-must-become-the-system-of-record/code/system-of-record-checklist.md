# Kayıt sistemi kontrol listesi

Yeni bir ajan, yalnızca depodan aşağıdakileri keşfedebilir mi?

- Hangi ürün geliştiriliyor?
- Uygulamanın kullanıcılar için ne yapması gerekiyor?
- Kod tabanı nasıl organize edilmiş?
- Uygulama nasıl başlatılır?
- Sağlık durumu nasıl kontrol edilir?
- Şu anda hangi iş devam ediyor?
- Hangi kalite standartları önemli?
