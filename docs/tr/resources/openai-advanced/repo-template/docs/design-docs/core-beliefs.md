# Çekirdek İnançlar

- Depo, ajanlar için kayıt sistemidir.
- `AGENTS.md` bir yönlendiricidir, ansiklopedi değil.
- Doğrulama kanıtı, özgüvenden daha önemlidir.
- Bir sınırlı görev, birçok yarım bırakılmış görevden iyidir.
- Tekrarlanan insan geri bildirimi, yeniden kullanılabilir harness
  kurallarına dönüşmelidir.
- Temizlik ve sadeleştirme, sevkiyatın parçasıdır, sonradan akla gelen
  şeyler değildir.
- Bir ajan bir gerçeği depoda keşfedemiyorsa, o gerçeği operasyonel
  olarak mevcut değil sayın.
