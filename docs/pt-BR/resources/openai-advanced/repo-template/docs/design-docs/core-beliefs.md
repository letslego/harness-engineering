# Princípios Fundamentais

- O repositório é a fonte da verdade para os agentes.
- `AGENTS.md` é um roteador, não uma enciclopédia.
- Evidências de verificação importam mais do que confiança.
- Uma tarefa bem delimitada é melhor do que várias tarefas pela metade.
- Feedback humano recorrente deve se transformar em regras reutilizáveis do harness.
- Limpeza e simplificação fazem parte da entrega, não são etapas posteriores.
- Se um agente não consegue descobrir um fato dentro do repositório, trate esse fato como operacionalmente indisponível.
