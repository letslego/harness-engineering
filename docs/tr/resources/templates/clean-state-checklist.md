# Temiz Durum Kontrol Listesi

- [ ] Standart başlatma yolu hâlâ çalışıyor.
- [ ] Standart doğrulama yolu hâlâ çalışıyor.
- [ ] Mevcut ilerleme, ilerleme günlüğüne kaydedilmiş.
- [ ] Özellik durumu, gerçekte geçenler ile doğrulanmamış olanları yansıtıyor.
- [ ] Belgelenmemiş, yarım kalmış adım yok.
- [ ] Sıradaki oturum manuel onarım olmadan devam edebilir.
