# Modelo de Repositório Avançado (Advanced Repo Template)

Copie este modelo inicial para um repositório real quando desejar uma superfície de documentação focada em agentes no estilo OpenAI, em vez de apenas um harness mínimo.

## Ordem de Cópia

1. Copie `AGENTS.md` e `ARCHITECTURE.md` para a raiz do repositório.
2. Copie toda a árvore `docs/`.
3. Preencha primeiro `docs/PRODUCT_SENSE.md`, `docs/QUALITY_SCORE.md` e `docs/RELIABILITY.md`.
4. Adicione seu primeiro plano ativo em `docs/exec-plans/active/`.
5. Mantenha os arquivos de ponto de entrada curtos e direcione os detalhes para os documentos vinculados.

## Para Que Este Modelo é Otimizado

- contexto local do repositório durável.
- divulgação progressiva em vez de um único arquivo de instrução gigante.
- ciclo de vida explícito do plano.
- rastreamento de qualidade ao longo do tempo.
- limites legíveis para agentes e humanos.

Trate cada arquivo aqui como um ponto de partida. Substitua espaços reservados, exemplos e comandos de amostra pelas especificidades reais do seu projeto antes de depender dele.
