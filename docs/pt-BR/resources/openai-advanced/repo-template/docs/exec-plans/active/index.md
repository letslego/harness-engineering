# Planos Ativos

Mantenha um arquivo Markdown por plano de execução ativo nesta pasta.

Padrão sugerido para nomes de arquivos:

- `AAAA-MM-DD-topico-curto.md`

Cada plano ativo deve estar atualizado o suficiente para que uma nova sessão de agente consiga retomar o trabalho apenas a partir do repositório.
