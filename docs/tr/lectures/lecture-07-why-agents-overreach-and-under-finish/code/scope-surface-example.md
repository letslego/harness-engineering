# Kapsam yüzeyi örneği

Görev:

- Electron bilgi tabanı uygulamasına dizinleme ekle

Kötü kapsam şekli:

- “Dizinlemeyi uygula”

Daha iyi kapsam şekli:

- İçe aktarılan belgeleri ayrıştır
- Belgeleri parçalara böl
- Parça meta verilerini kalıcı hale getir
- Dizinleme durumunu arayüzde göster
- Yeniden dizinleme eylemi ekle
