# Anti-Patterns em Arquivos de Instrução

* Colocar todo o conhecimento do repositório em um único arquivo
* Repetir a mesma regra em múltiplos lugares
* Manter regras obsoletas que ninguém revisa
* Escrever instruções condicionais tão específicas que raramente se aplicam
* Embutir manuais longos de ferramentas no contexto inicial
