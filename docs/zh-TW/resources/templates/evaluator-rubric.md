# 評審評分表

在實作完成後、正式接受前，使用這張評分表進行評審。

| 維度 | 問題 | 分數（0-2） | 備註 |
| --- | --- | --- | --- |
| 正確性 | 實作出的行為是否符合要求的功能？ |  |  |
| 驗證 | 要求的檢查是否真的執行，且附有證據？ |  |  |
| 範圍紀律 | 這個工作階段是否留在選定功能的範圍內？ |  |  |
| 可靠性 | 結果在重啟或重跑後，是否仍能運作而不需修復？ |  |  |
| 可維護性 | 程式碼與文件是否足以讓下一個工作階段接手？ |  |  |
| 交接準備度 | 新的工作階段是否能只依靠儲存庫工件繼續工作？ |  |  |

## 判定

- Accept
- Revise
- Block
