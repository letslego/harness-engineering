# Değerlendirici rubriği örneği

Her boyut için 1-5 arası puanlama kullanın:

- Dayanak (Grounding): cevaplar içe aktarılan kaynaklara açıkça bağlı mı?
- Atıf kalitesi: kaynak referansları görünür ve belirgin mi?
- İşlevsellik: kullanıcı soru-cevap akışını tamamlayabiliyor mu?
- Ürün tutarlılığı: iş akışı entegre hissettiriyor mu?
