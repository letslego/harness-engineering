# Exemplo de Superfície de Escopo

Tarefa:

- Adicionar indexação ao aplicativo de conhecimento em Electron

Formato de escopo inadequado:

- “Implementar indexação”

Formato de escopo melhor:

- Processar documentos importados
- Dividir documentos em blocos (*chunks*)
- Persistir metadados dos blocos
- Expor o status da indexação na interface
- Adicionar uma ação de reindexação
